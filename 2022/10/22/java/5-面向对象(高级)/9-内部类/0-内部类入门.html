一、基本介绍

1.简介

类的五大成员

  • 属性
  • 方法
  • 构造器
  • 方法块
  • 内部类

2.根据定义的位置分类：

如果定义类在局部位置(方法中/代码块) :

(1) 局部内部类 ==(2) 匿名内部类==

定义在成员位置

(1) 成员内部类 (2) 静态内部类

3.基本使用

package com.hspedu.innerclass;
public class InnerClass01 { //外部其他类
public static void main(String[] args) {
}
}
class Outer { //外部类
private int n1 = 100;//属性
public Outer(int n1) {//构造器
this.n1 = n1;
}
public void m1() {//方法
System.out.println("m1()");
}
{//代码块

}

}
