一、==

  1. 可以判断基本类型和引用类型

  2. 判断基本类型

    1. 比较的是值是否相等

  3. 判断引用类型

    1. 判断内存地址是否相同/引用是否指向同一个对象

    2. object的equal方法的基本组成

      1. public boolean equals(Object obj) {
        return (this == obj);
    }

二、equal

  1. equal 方法只能比较引用类型

  2. object 类的 equal 方法

    1.          public boolean equals(Object obj) {
                 return (this == obj);
             }

  3. Object 的子类 String 和 integer 都重写了 object 中的 equal 方法

    1. //String进行判断

            String anotherString = (String)anObject;

            if (n == anotherString.value.length) {
